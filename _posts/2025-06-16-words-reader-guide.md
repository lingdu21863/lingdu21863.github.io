## 单词朗读器介绍

本站提供了一个在线英语单词朗读器，支持多种词库和自定义设置。

### 功能特点

- **多种词库**：支持初中、高中、四级、六级、考研、托福、SAT 等 7 种词库
- **单词乱序**：每次加载词库后自动打乱单词顺序
- **语音朗读**：使用 Web Speech API 实现语音朗读
- **自动播放**：支持自动播放模式
- **语速调节**：支持 0.5x - 2.0x 的语速调节

### 使用方法

1. 访问 [单词朗读](/words/) 页面
2. 选择词库
3. 点击单词卡片进行朗读
4. 使用快捷键控制播放

### 快捷键

| 按键 | 功能 |
|------|------|
| ↑ | 上一个单词 |
| ↓ | 下一个单词 |
| 空格 | 播放/暂停 |

### 学习建议

1. 每天坚持学习一定数量的单词
2. 结合例句理解单词用法
3. 定期复习已学单词

祝学习愉快！
